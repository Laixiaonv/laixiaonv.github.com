CSS选择器（加强版）

  • 属性选择器
    • [attr*=val] ：元素用attr表示的值中包含用val指定的字符
      例：a[src*="abc"] 选择其 src 属性中包含 “abc” 子串的每个<a>元素。
      a[href*="e3.org"] 选择href属性中包含“e3.org”的<a>元素。
    • [attr^=val] ：开头字符为val的元素
      例：a[src^="abc"] 选择其 src 属性中开头为“abc” 字符串开头的每个<a>元素。
      a[href^="e3.org"] 选择href属性中以“e3.org”字符串开头的<a>元素。
    • [attr$=val] ：结尾字符为val的元素 [属性$=尾字符]
      例：若id="section1-2-1" 则[id$=\-1]（因为连接符‘-’前必须加上‘\’）
    • [attr|=val] ：attr属性开头字符为val的元素
  • 伪类选择器
    • first-child ：属于父元素的第一个元素
      例：ul的第一个子元素 ul>li:first-child应先找到所有子元素，再从子元素中选第一个
    • last-child ：属于父元素的最后一个元素
      用法同上
    • nth-child(n)："n"是指第几个，注意是从1开始
      如：ul li:nth-child(1) => 选中父元素内第一个li子元素
  • 锚伪类
    • link：未访问
    • active ：选择活动链接
    • visited ：已被访问
    • hover ：鼠标悬浮（不仅适用于链接）
  • 伪元素
    • first-line：向某个元素的第一行文字使用
    • first-letter：首字母或第一个字
    • before：再某个元素之前插入一些内容
    • after：在某个元素之前插入一些内容
  • 否定伪类选择符E:not(){} 匹配不含有s选择符的元素E
    p:not(.a){}：除了class = "a"的所有p标签元素

各种变换

  • transform
    • scale:放大/缩小
      transform:scale()
      scaleX():x轴方向变化
      scaleY():y轴方向变化
    • translate：2D平移
      transform:translate() 括号中写两个值时，要用逗号隔开
      translateX():x轴方向变化
      translateY():y轴方向变化
    • skew:扭曲 参数为度数，如-45deg
      transform:skew(x,y) 只写一个参数时，y方向默认为0
      skewX():x轴方向变化
      skewY():y轴方向变化
    • transform-origin:改变中心点
      left：指定原点的横坐标为left
      center①：指定原点的横坐标为center
      right：指定原点的横坐标为right
      top：指定原点的纵坐标为top
      center②：指定原点的纵坐标为center
      bottom：指定原点的纵坐标为bottom
      参数还可以设为数值，如transform-origin：50px 50px
      同时也可以是百分比，如transform-origin：50% 50%
    • matrix(a,b,c,d,e,f) a:x轴缩放
      b:x轴倾斜角度
      c:y轴倾斜角度
      d:y轴缩放
      e:水平偏移
      f:垂直偏移
      推荐一个写的很详细的博客：点我点我点我
      这个东西真的太难以用言语表达，我们需要借助一下外来的力量
  • transaction:过渡效果
    • transaction-property:设置属性是否参与过渡，可以写all，或者是单独某个样式
      transaction-property:all
      transaction-property:width,height,color
    • transaction-duration:设置过渡时间
    • transaction-timing-function:动画类型
      linear:线性
      ease-in：慢 -> 快
      ease-out：快 -> 慢
      ease：平滑
      ease-in-out：慢 -> 快 -> 慢
    • transaction-delay:设置延迟时间
      • 复合型：① ② ③ ④
  • 动画属性animation(@keyframes + 名字)，同样也具有复合型
    • animation-duration：time 指定对象动画的持续时间
    • animation-timing-function：同transaction过渡效果
    • animation-delay：设置对象动画延迟时间
    • animation-iteration-count：设置动画的具体循环次数，若为“infinite”，则为无限循环
  • 边框渲染(box-sizing)
    • content-box:标准盒模型，padding和margin不包含在width和height之中
    • border-box：怪异盒模型，包含在width和height之间
