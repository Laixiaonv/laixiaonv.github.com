特点

  • HTML5 文档类型
    Bootstrap 使用到的某些 HTML 元素和 CSS 属性需要将页面设置为 HTML5 文档类型。在你项目中的每个页面都要参照下面的格式进行设置。
    <!DOCTYPE html>
    <html lang="zh-CN">
    ...
    </html>
  • 移动设备优先

文本元素

  • 标题：除了最常用h1~h6之外，还添加了副标题样式(小号文本)，(<small> 标签或赋予 .small 类的元素)
    <h3>h3. Bootstrap heading <small>Secondary text</small></h3>

    h3. Bootstrap headingSecondary text

  • <mark> ：标签
  • <del> ：被删除文本
    <s> ：无用文本(其实我觉得和del标签所显示的样式没什么太大的区别)
  • <ins> ：插入文本
    <u> ：带下划线文本(和插入也差不多)
  • 对齐方式：可以直接添加class样式
    class="text-left"
    class="text-right"
    class="text-center"
    class="text-justify" (默认样式)
    class="text-nowrap" (不自动换行)

代码

  • 内联代码<code>:包裹内联样式代码片段
  • 标记键盘输入内容
    Ctrl+Shift Alt
  • 多行代码块使用<pre>标签
  • <var>标签：用来标记变量
    y = 15x+20//普通写法
    y = 15x + 20 //<var>y</var> = 15<var>x</var> + 20
  • 程序输出<samp>

表格

  • 在table标签中加入.table类，反正就会好看很多
  • 条纹状表格（也就是奇偶行颜色不同），在<tbody>中加入.table-striped类
    要注意的是，在该情况下，必须要有完整的表格结构，即thead、tbody， 否则表格的第一行标题栏样式会有问题
  • .table-bordered:为表格和其中上的每个单元格添加边框
  • .table-hover:鼠标悬停
