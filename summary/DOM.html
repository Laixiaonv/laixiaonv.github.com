DOM（文档对象模型）

用于HTML的应用程序接口，有三个级别；通过DOM可以去改变文档；一共有12种类型
  • DONE对象：一种用来表示HTML元素的原始数据类型，即是所有节点的基类型，所有节点都继承自它，所以所有节点都有一些共同的方法和属性
    节点对象代表文档树中一个单独的节点（HTML文档中所有的内容都是节点，如标签、空白的地方等）
    • childNodes属性：返回节点的子节点（NodeList对象，极其像数组）
    • appendChild()方法：方法可以在节点的childNodes的末尾增加一个节点
    • removeChild()方法：方法指定元素的某个指定的子节点,以 Node 对象返回被删除的节点，如果节点不存在则返回 null。 
      									< ul id="nav" >
										< li >青菜< /li >
										< li >水果< /li >
										< li >蛋糕< /li >
									< /ul >
									var list = document.getElementById("nav");
									list.removeChild(list.childNodes[0]);
								=> 输出结果为“水果 蛋糕”
  • document对象
    • document.getElementById():传入ID，得到元素节点
    • document.getElementsByClassName():返回文档中所有指定类名的元素集合，作为NodeList对象，可以使用 NodeList 对象的 length 属性来确定指定类名的元素个数，并循环各个元素来获取你需要的那个元素
    • document.getElementsByTagName():根据标签取得元素
    • document.getElementsByName():返回带有指定名称的对象集合
    • createElement():创建元素节点，可返回一个element对象 
      								var name = document.getElementById("id");
								var tag = document.createElement("input");
  • Element对象的方法
    • setAttribute() 方法:创建或改变某个新属性 
      									var x = document.getElementById("text");
									x.setAttribute("class","detail");
									x.setAttribute("style","color:red;font-size:16px;");
    • getAttribute() 方法:获取指定元素的属性值
    • appendChild() 方法:加入节点（加在最后一个位置） 
      								var div=document.createElement("div");			//新建一个div元素节点
								document.body.appendChild(div);			//把div元素节点添加到body元素节点中成为其子节点，但是放在body的现有子节点的最后
