git上传项目到GitHub

  • 安装，配置环境变量
  • git常用命令
    • git verson查看git版本信息
    • git help:git帮助命令
  • 管理项目常用命令
    • git init:初始化仓库
    • git add yourfilesname.yourfilestype(例如：test.txt/index.html...and so on...):提交单个文件
    • git add .:将所有文件和文件夹提交到待提交文件的队列中
    • git commit :提交到版本库中
    • -m :添加提交到版本库的版本描述信息
  • 备份本地项目到云端：
    • 注册登录云端git服务器
    • 在云端创建一个备份项目
    • 克隆云端项目到本地 git clone httpsaddress(例如：https://www.baidu.com/moxing.git/git:git.....git)
    • 复制本地项目到git 项目中
    • cd 项目目录
    • git add .:提交所有文件/文件夹到待提交阵列
    • git commit -m "描述":提交到版本库
    • git push origin master:提交到云端版本库
